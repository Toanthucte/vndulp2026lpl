# IMAGE MAP A

## Quy ước chuẩn

Cấu trúc chuẩn để map ảnh vào app:

`assets/image/<Nhóm bệnh>/<Mã bệnh-Tên bệnh>/<Mã ảnh-Mô tả>.png`

Ví dụ:

`assets/image/A-Các Bệnh Về Đầu - Mặt - Cổ - Gáy/A2-Nhức nửa đầu/A2a-...png`

## Canonical source

Ưu tiên dùng ảnh trong các thư mục con `A1...A7`.
Không dùng các ảnh legacy dạng `Picture19...Picture32...` ở thư mục gốc nhóm `A` để map vào app.

## Ghi chú

- Nhóm `A6` và `A7` đã được chuẩn hóa lại tên file theo đúng mã ảnh.
- Nhóm `A5` còn 1 file có khoảng trắng thừa sau mã `A5a-b-c-d-`; app hiện đã map đúng theo tên thật trên đĩa.
- Khi có thời gian, nên đổi tên vật lý file `A5a-b-c-d- ...` thành `A5a-b-c-d-...` để đồng nhất hoàn toàn.
